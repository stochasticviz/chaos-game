Play around with the Chaos game!
Write your own code to calculate where the next point should be. You can use the coordinates of the current point, currentPoint, and an matrix, targetPoints, which contains the coordinates of each target vertex. The default code simply picks one of those targets at random and moves the current point halfway there. But you can change that using functions, operators, or whatever else you can find in MathJS. All you must do is set the nextPoint variable to a array of two numbers.

Example: 
nextPoint = currentPoint + 1;
Example: 
nextTargetIndex = math.randomInt(1, targetPointsLength+1);
_nextTarget = targetPoints[nextTargetIndex, :];
nextX = (currentX + _nextTarget.x) / 2 + (currentY - _nextTarget.y) / 4;
nextY = (currentY + _nextTarget.y) / 2 - (currentX - _nextTarget.x) / 4;
Example: 
nextTargetIndex = math.randomInt(1, targetPointsLength+1);
_nextTarget = targetPoints[nextTargetIndex, :];
midpoint = (currentPoint + _nextTarget) / 2;
rotation = [[cos(pi/4), -sin(pi/4)], [sin(pi/4), cos(pi/4)]];
nextPoint = midpoint * rotation ;
Example: 
nextTargetIndex = math.randomInt(1, targetPointsLength+1);
_nextTarget = targetPoints[nextTargetIndex];
midpoint = [(currentX + _nextTarget.x)/2, (currentY + _nextTarget.y)/2];
rotation = [[cos(pi/4), -sin(pi/4)], [sin(pi/4), cos(pi/4)]];
